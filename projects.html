Raspberry Pi Webserver

WebServer

This project was a personal project. I wanted to learn how to use linux command line and hosting a website from a raspberry pi. I have spent a long time reading up on how linux works and how to use it efficiently. I run my webserver off the debian operating system with no UI. I used the command lines to download Nginx and Git so my project can handle the requests/responses my website will recieve and to download the files I needed for my website on my GitHub account.
